# Creative Style - 创意风格

## Design Aesthetic

创意活泼风格，强调视觉冲击力、色彩丰富和创意表达。使用大胆的配色、多样的装饰元素和灵活的布局，适合创意提案、品牌宣传、活动策划等需要吸引眼球的场合。

核心理念:
- 视觉冲击: 使用大胆的配色和排版
- 创意表达: 灵活的布局和多样的视觉元素
- 情感共鸣: 通过色彩和设计传达情感

## Background

- 颜色: 多样化，可以是纯色、渐变或图案
- 纹理: 可选纸张纹理、几何图案、插画背景
- 渐变: 常用，多色渐变或双色渐变

常见背景:
- 暖色渐变: 从 #FF6B6B 到 #FFE66D
- 冷色渐变: 从 #4ECDC4 到 #556270
- 彩虹渐变: 多色渐变
- 纯色背景: #FF6B6B, #4ECDC4, #FFE66D 等

## Typography

### Primary Font (Headlines)
- 字体: 黑体 (Heiti) / Microsoft YaHei Bold / 创意字体
- 粗细: Bold (700) 或 Extra Bold (800)
- 大小: 52pt (标题), 36pt (小标题)
- 颜色: 多样化，根据背景调整

### Secondary Font (Body)
- 字体: 宋体 (SimSun) / Microsoft YaHei
- 粗细: Regular (400)
- 大小: 22pt (正文), 18pt (注释)
- 颜色: 深灰 (#2D3748) 或纯白 (#FFFFFF)

## Visual Elements

- Logo: 创意 Logo，位置灵活
- 页码: 创意设计，可以是图形化的页码
- 装饰元素: 丰富多样
  - 几何图形 (圆、三角、波浪线)
  - 插画元素
  - 图标
  - 色块
  - 手绘元素
  - 贴纸效果
- 图标: 扁平化或手绘风格，多彩
- 边框: 可以是粗线条、虚线、波浪线
- 色块: 多样化，可以是渐变、纯色、半透明

## Style Rules

### Do (应该保持)
- 使用大胆的配色组合
- 保持视觉层级清晰
- 使用多样的装饰元素增加趣味性
- 保持元素对齐 (即使布局灵活)
- 使用渐变、阴影等视觉效果
- 保持整体风格统一 (即使元素多样)
- 使用插画、图标等增强视觉表达

### Don't (禁止改变)
- 不添加幻灯片编号、页脚或 Logo (除非模板原有)
- 不使用过于严肃的配色 (如纯黑白灰)
- 不使用过于简单的布局 (要有创意)
- 不使用单一的装饰元素
- 不忽视可读性 (即使追求创意)
- 不使用过多的颜色导致混乱 (控制在 5-7 种主色)
- 不使用与主题不符的装饰元素

## Best For

- 品牌宣传
- 活动策划
- 产品发布 (面向年轻群体)
- 社交媒体内容
- 营销活动
- 设计工作室介绍
- 创业路演
- 文化活动宣传
- 教育培训 (面向儿童/青少年)
