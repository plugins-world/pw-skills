# Minimal Style - 简约风格

## Design Aesthetic

极简主义设计风格，强调留白、简洁和视觉冲击力。通过精简的元素和大胆的排版创造强烈的视觉效果，适合产品发布、设计展示等需要突出重点的场合。

核心理念:
- 少即是多: 只保留最必要的元素
- 留白为王: 大量留白突出核心内容
- 视觉冲击: 通过对比和排版创造视觉焦点

## Background

- 颜色: 纯白色 (#FFFFFF) 或深灰色 (#1F2937)
- 纹理: 无
- 渐变: 无

## Typography

### Primary Font (Headlines)
- 字体: 黑体 (Heiti) / Microsoft YaHei Bold
- 粗细: Bold (700) 或 Extra Bold (800)
- 大小: 60pt (主标题), 40pt (小标题)
- 颜色: 纯黑 (#000000) 或纯白 (#FFFFFF)

### Secondary Font (Body)
- 字体: 宋体 (SimSun) / Microsoft YaHei Light
- 粗细: Light (300) 或 Regular (400)
- 大小: 20pt (正文), 16pt (注释)
- 颜色: 深灰 (#374151) 或中灰 (#6B7280)

## Visual Elements

- Logo: 极简设计，通常只有图标或文字，位置不固定
- 页码: 极小字号，位置不固定，可选
- 装饰元素: 极少，只使用简单的线条或几何图形
- 图标: 线条图标，极简风格
- 边框: 极少使用，如有则为细线条 (1px)
- 色块: 很少使用，如有则为纯色

## Style Rules

### Do (应该保持)
- 保持大量留白 (至少 10-15% 边距)
- 使用极简的排版，突出核心内容
- 保持黑白灰为主的配色
- 使用大字号标题创造视觉冲击
- 保持元素对齐和网格系统
- 使用简单的几何图形 (圆、方、线)
- 保持信息层级极其清晰

### Don't (禁止改变)
- 不添加幻灯片编号、页脚或 Logo (除非绝对必要)
- 不使用多余的装饰元素
- 不使用复杂的颜色系统 (保持黑白灰)
- 不使用渐变、阴影等效果
- 不使用复杂的图案或纹理
- 不填满整个页面，保持留白
- 不使用多种字体
