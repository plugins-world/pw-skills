# PPT 风格库

本目录包含预定义的 PPT 风格模板，用于快速生成符合特定风格的 PPT 页面。

## 使用方法

1. 选择合适的风格模板
2. 复制风格定义到提示词的【风格约束】部分
3. 根据实际模板微调参数
4. 生成图片

## 风格模板列表

### corporate.md - 商务风格
- 适用场景: 企业汇报、商务演示、年度总结
- 特点: 专业、简洁、清晰
- 主色调: 蓝色系

### minimal.md - 简约风格
- 适用场景: 产品发布、设计展示、创意提案
- 特点: 极简、留白多、视觉冲击力强
- 主色调: 黑白灰

### technical.md - 技术风格
- 适用场景: 技术分享、开发文档、架构设计
- 特点: 清晰、逻辑性强、信息密度高
- 主色调: 深色系

### creative.md - 创意风格
- 适用场景: 创意提案、品牌宣传、活动策划
- 特点: 活泼、色彩丰富、视觉元素多
- 主色调: 多彩

## 风格定义结构

每个风格文件包含以下标准化章节:

1. **Design Aesthetic** - 整体设计美学描述
2. **Background** - 背景颜色和纹理
3. **Typography** - 字体系统
   - Primary Font (Headlines) - 标题字体
   - Secondary Font (Body) - 正文字体
4. **Color Palette** - 完整的颜色系统表格
5. **Visual Elements** - 视觉元素列表
6. **Style Rules** - 风格规则
   - Do (应该做的)
   - Don't (不应该做的)
7. **Best For** - 最适合的使用场景

## 自定义风格

如果需要创建新的风格模板:

1. 复制现有风格文件作为模板
2. 按照标准结构填写各个章节
3. 确保颜色系统使用精确的 Hex 值
4. 明确列出风格约束规则
5. 提供使用场景说明

## 注意事项

- 所有颜色必须使用精确的 Hex 值
- 字体大小要根据实际模板调整
- 风格约束要明确、具体、可执行
- 禁止项要清晰列出，避免歧义
