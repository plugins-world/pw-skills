# 提示词模板

## 用途

此文件用作图像生成的提示词模板。复制到项目目录后，可以按需要修改。

## 提示词示例

### 示例 1：动物场景

```
## 提示词

一只可爱的柴犬在樱花树下睡觉，水彩风格，柔和的粉色调，春天的氛围，毛发蓬松，细节丰富，4K 分辨率。
```

### 示例 2：风景场景

```
## 提示词

一片宁静的湖泊，湖水清澈见底，周围环绕着高大的松树，远处有雪山，日落时分，金色的阳光洒在雪山上，油画风格。
```

### 示例 3：人物场景

```
## 提示词

一个穿着古风服饰的少女，站在竹林中，手持一把油纸伞，微风轻轻吹动头发，水墨风格，淡雅的色调，意境悠远。
```

## 提示词格式规范

### 结构建议

1. 使用 `## 提示词` 标题开始
2. 主体描述
3. 风格说明
4. 颜色和氛围
5. 细节描述
6. 技术参数

### 编写技巧

- **详细描述**：场景、主体、风格、颜色、光照等
- **避免歧义**：不要使用模糊词汇
- **具体词汇**：使用具体的名词和形容词
- **顺序**：主体 → 动作 → 场景 → 风格 → 细节

### 常见错误

避免使用以下类型的提示词：

1. **太过模糊**："一幅漂亮的画"
2. **包含矛盾元素**："一只透明的柴犬"
3. **违规内容**：包含暴力、色情等禁止内容
