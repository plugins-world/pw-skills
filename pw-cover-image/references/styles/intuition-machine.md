# intuition-machine

技术简报风格, 带有陈旧纸张和双语标签

## 色彩方案

- 主色: 深栗色 (#5D3A3A), 青色 (#2F7373)
- 背景: 陈旧奶油 (#F5F0E6), 微妙的纸张纹理, 带有轻微折痕
- 强调色: 温暖棕 (#8B7355), 栗色 (#722F37), 深炭灰 (#2D2D2D)

## 视觉元素

- 等距 3D 技术插图或扁平 2D 图表
- 双语标注标签 (英文 + 中文)
- 褪色的主题背景图案
- 所有元素上的清晰黑色轮廓
- 分割或三联画布局
- 底部的关键引用框

## 字体排版

- 深栗色的粗体展示字体, 标题全大写, 双语标签

## 最适合

技术解释, 学术演示, 双语受众, 知识文档
